# About Palms Gig
Palms Gig is a digital service and gig-based platform that connects individuals
and businesses with skilled and unskilled task performers. The company
provides a marketplace where users can create, post, and manage tasks, while
other users complete these tasks in exchange for agreed payments.
Palms Gig facilitates task matching, communication, and payment processing,
enabling flexible work opportunities and efficient task execution across various
sectors.

## üìñ Cloudinary URL Parameters Reference

| Parameter | Effect | Example |
|-----------|--------|---------|
| `f_auto` | Auto format (WebP/AVIF) | Reduces size 30-80% |
| `q_auto` | Auto quality | Optimal compression |
| `w_X` | Width in pixels | `w_400` = 400px wide |
| `h_X` | Height in pixels | `h_300` = 300px tall |
| `c_scale` | Scale proportionally | Maintains aspect ratio |
| `c_fill` | Fill & crop | Exact dimensions |
| `dpr_auto` | Device pixel ratio | Retina support |

**Combine parameters with commas:** `f_auto,q_auto,w_800`
